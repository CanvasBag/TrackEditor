Detailed Description

Specifies an addition, change, or deletion to a KML resource that has previously been retrieved via NetworkLink.

OGC KML 2.2 Section 13.3

Update does not affect the KML resource itself; rather it updates its representation within the earth browser only.

All KML objects within an update context, that is a grandchild of the Update element, shall have a KmlObject.TargetId that identifies the object to be updated, and shall not have an KmlObject.Id.
