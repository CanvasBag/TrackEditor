Detailed Description

Specifies the position and orientation of a virtual camera.

OGC KML 2.2 Section 14.2

Can be used to specify views of the earth or of objects in space.

◆ Longitude

double SharpKml.Dom.Camera.Longitude
getset

Gets or sets the geodetic longitude of the virtual camera (eye point).

The angular distance is in decimal degrees, relative to the Prime Meridian. Values west of the Meridian range from −180 to 0 degrees. Values east of the Meridian range from 0 to 180 degrees.

◆ Roll

◆ Tilt

double SharpKml.Dom.Camera.Tilt
getset

Gets or sets the rotation, in decimal degrees, of the camera around the X axis.

A value of 0 indicates that the view is aimed straight down toward the earth (the most common case). A value of 90 indicates that the view is aimed towards the horizon. Values greater than 90 indicate that the view is pointed up into the sky. Values are clamped at +180 degrees.
