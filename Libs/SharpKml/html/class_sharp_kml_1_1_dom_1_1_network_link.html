Detailed Description

References a KML resource on a local or remote network.

OGC KML 2.2 Section 9.12

May be used in combination with Region elements to efficiently load and display large datasets.

Property Documentation

◆ FlyToView

bool SharpKml.Dom.NetworkLink.FlyToView
getset

Gets or sets whether to adjust the geographic view upon activation.

A value of false indicates that the geographic view shall remain unchanged.

A value of true indicates that the geographic view shall be displayed according to the AbstractView specified by either:

  • a NetworkLinkControl
  • a child Feature of Kml

if they exist in the referenced KML resource. The AbstractView of the NetworkLinkControl shall take precedence over the AbstractView of the Feature if they both exist. If neither exists then the view shall remain unchanged.

◆ RefreshVisibility

bool SharpKml.Dom.NetworkLink.RefreshVisibility
getset

Gets or sets the visibility of any Features within the referenced KML resource.

A value of false shall leave the visibility of any referenced Features in the geographic view within the control of the earth browser user.

A value of true shall require any referenced Features to be visible within the geographic view whenever they are refreshed.
