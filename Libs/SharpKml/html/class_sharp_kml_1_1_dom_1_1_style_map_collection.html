Detailed Description

Specifies a mapping between two Styles using a key/value pair that maps a mode to the predefined Feature.StyleUrl.

OGC KML 2.2 Section 12.3

A StyleMap may be used to provide separate normal and highlighted styles for a Placemark. StyleMap should contain two Pair objects, one with a Pair.State value of StyleState.Normal and the other with a value of StyleState.Highlight.
