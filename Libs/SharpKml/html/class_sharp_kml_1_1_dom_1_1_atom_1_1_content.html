Detailed Description

Represents information about or links to the content of an entry.

RFC 4287 Section 4.1.3 (see http://atompub.org/rfc4287.html) This is not part of the OGC KML 2.2 standard.

Property Documentation

◆ MediaType

string SharpKml.Dom.Atom.Content.MediaType
getset

Gets or sets the MIME media type of the content.

The value of may be one of "text", "html", or "xhtml". Failing that, it must conform to the syntax of a MIME media type, but must not be a composite type [RFC 4288].

◆ Source

Gets or sets an URI reference to retrieve the content from.

Remote content is allowed to be ignored or presented in a different manner than local content.
