Detailed Description

Provides common properties for Icon and Link.

This is not part of the KML specification but is declared in the XSD. Because it is not part of the specification, this class should not be used; an instance of Icon or Link should be used instead.

◆ HttpQuery

string SharpKml.Dom.LinkType.HttpQuery
getset

Gets or sets a value used to specify any additional query parameters not related to the geographic view.

The following query parameters may be used:

  • [clientVersion] Version of earth browser client.
  • [kmlVersion] Version of requested kml.
  • [clientName] Name of earth browser client.
  • [language] Language preference of the earth browser client.

◆ ViewBoundScale

double SharpKml.Dom.LinkType.ViewBoundScale
getset

Gets or sets the value to scale all bounding box parameters.

A value less than 1 specifies to use a geographic area less than the current geographic view. A value greater than 1 specifies to use a geographic area greater than the current geographic view. The value shall be positive.

◆ ViewFormat

string SharpKml.Dom.LinkType.ViewFormat
getset

Gets or sets the format of a query string related to view parameters that is appended to the BasicLink.Href before the resource is fetched.

The following query parameters may be used:

  • [lookatLon], [lookatLat] Longitude and latitude of the point that LookAt is viewing.
  • [lookatRange], [lookatTilt], [lookatHeading] Values used by the LookAt element (the Range, Tilt, and Heading properties).
  • [lookatTerrainLon], [lookatTerrainLat], [lookatTerrainAlt] Point on the terrain in decimal degrees/meters that LookAt is viewing.
  • [cameraLon], [cameraLat], [cameraAlt] Decimal degrees/meters of the eye point for the camera.
  • [horizFov], [vertFov] Horizontal, vertical field of view for the camera.
  • [horizPixels], [vertPixels] Size in pixels of the geographic view.
  • [terrainEnabled] Indicates whether the geographic view is showing terrain.
  • [bboxWest], [bboxSouth], [bboxEast], [bboxNorth] Bounding box limits matching the OGC Web Map Service (WMS) bounding box specification.

◆ ViewRefreshMode

ViewRefreshMode SharpKml.Dom.LinkType.ViewRefreshMode
getset

Gets or sets how the link is refreshed when the geographic view changes.

If ViewRefreshTime is specified then ViewRefreshMode should be set to Dom.ViewRefreshMode.OnStop.

◆ ViewRefreshTime

double SharpKml.Dom.LinkType.ViewRefreshTime
getset

Gets or sets the number of seconds to wait before refreshing the geographic view after camera movement stops.

This applies when ViewRefreshMode is set to Dom.ViewRefreshMode.OnStop. The value shall be positive.
