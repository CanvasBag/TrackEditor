Detailed Description

Represents a KML AbstractTimePrimitiveGroup.

OGC KML 2.2 Section 15.1

All DateTime instances in TimePrimitive and its descendants should be in the UTC form (i.e. have their System.DateTime.Kind set to System.DateTimeKind.Utc).
