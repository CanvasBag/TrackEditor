Detailed Description

Defines a closed line string that should not cross itself.

OGC KML 2.2 Section 10.5

If LinearRing is used to define a boundary for a Polygon then Extrude, Tessellate and LinearRing.AltitudeMode should not be specified.

◆ Extrude

bool SharpKml.Dom.LinearRing.Extrude
getset

Gets or sets whether to connect a geometry to the ground.

The geometry is extruded toward the Earth's center of mass. To extrude a geometry, AltitudeMode shall be either Dom.AltitudeMode.RelativeToGround or Dom.AltitudeMode.Absolute, and the altitude component should be greater than 0 (that is, in the air).
