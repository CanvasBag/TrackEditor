Detailed Description

Represents a Kmz archive, containing Kml data and associated files.

The entire Kmz archive (in its compressed state) will be held in memory until a call to KmzFile.Dispose is made.

KmlFile SharpKml.Engine.KmzFile.GetDefaultKmlFile ( )

Loads a default KmlFile inside this archive.

Returns
A KmlFile representing the default KML file in the specified KMZ archive or null if no KML data was found.

This method checks for duplicate Id's in the file and throws an exception if duplicate Id's are found. To enable duplicate Id's use the SharpKml.Base.Parser class and pass the root element to KmlFile.Create.

string SharpKml.Engine.KmzFile.ReadKml ( )

Extracts the default Kml file from the archive.

Returns
A string containing the Kml content if a suitable file was found in the Kmz archive; otherwise, null.

This returns the first file in the Kmz archive table of contents which has a ".kml" extension. Note that the file found may not necessarily be in the root directory.
